## 🧠 Task 02 – AI Content Marketing Using UGC Ads

### 📌 Task Overview
This project demonstrates how **Prompt Engineering** can be used to create high-converting **UGC (User-Generated Content) ad scripts** for a skincare product.

- **Track:** Prompt Engineering (PE)  
- **Product:** GlowPure Vitamin C Brightening Serum  
- **Brand Type:** D2C Skincare Brand  
- **Platforms:** Instagram Reels, Facebook Ads, YouTube Shorts  
- **Tools Used:** Claude AI, Google Docs, GitHub  

---

## 🎯 Objective
To generate engaging, conversion-focused ad content using AI, including:
- Scroll-stopping hooks  
- UGC-style ad scripts  
- Platform-specific captions  

---

## 🧠 Prompt Engineering Strategy
Prompts were designed using:
- Role-based prompting (AI as ad copywriter)  
- Target audience specification (women 20–35)  
- Platform-specific instructions  
- Tone guidance (authentic, relatable)  
- Structured output formats  

---

## 🎣 Scroll-Stopping Hooks

Examples:

- "I applied this every night for 14 days…"  
- "Tired of wearing full coverage foundation?"  
- "I spent ₹8,000… then tried this for ₹799."  
- "Everyone is using this serum now…"  
- "This is my skin after 3 weeks — no filter."  

---

## 📱 UGC Ad Scripts

### Instagram Reels (30 sec)
- Problem → Story → Product → Proof → CTA  
- Casual, relatable tone  
- Includes before/after storytelling  

---

### Facebook Ad
- Problem-Solution-CTA format  
- Key benefits:
  - 15% Vitamin C  
  - Hydration support  
  - Dermatologist-tested  

---

### YouTube Shorts
- Educational + testimonial format  
- Includes:
  - Statistic-based hook  
  - Product explanation  
  - Personal results  

---

## 💬 Social Media Captions

- **Instagram:** Emotional + engaging with emojis  
- **Facebook:** Informational and trust-building  
- **YouTube:** SEO-optimized description  

---

## 📊 Key Learnings

- UGC ads perform better due to authenticity  
- AIDA framework improves conversions  
- Platform-specific prompts improve output quality  
- Structured prompts give consistent results
